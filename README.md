# TaskFlow
A comprehensive UI/UX case study and companion landing page for **TaskFlow**, an intuitive mobile productivity and task management application. This project was developed as part of a Human-Computer Interaction (HCI) course to explore user-centered design paradigms, iterative prototyping, and responsive front-end implementation.

* **Live Landing Page:** [Visit Website](https://hayaturkii.github.io/TaskFlow/)

## Project Overview
TaskFlow is a conceptual productivity application designed to help users organize tasks, manage projects, and stay focused throughout their daily routines.

This project consists of two complementary components:
* UI/UX Prototype
A high-fidelity interactive mobile application prototype designed in Figma. The prototype demonstrates user flows, interface interactions, and visual design decisions informed by Human-Computer Interaction (HCI) principles.
<br> **Note:** The mobile application itself was not developed. This project focuses on the design, research, and prototyping process.
* Landing Page Website
A responsive front-end marketing website built to showcase the TaskFlow concept, highlight its features, and present the application in a professional and engaging format.

## Key Features Demonstrated
* **Task Management:** Create and organize tasks, subtasks, reminders, and due dates.
* **Focus Mode:** A distraction-free workspace designed to prioritize essential daily activities.
* **Project Organization:** Track project progress and milestones through structured workflows.
* **Tiered Plans:** Conceptual Free, Pro, and Enterprise plans featuring productivity analytics and AI-powered suggestions.
